B.Tech. First Year, Semester-I (Common to all branches)

Course Name: Web Designing

Course Objective

  • Introduce students to the fundamentals of web technologies and the full-stack development environment.Familiarize students with modern tools like VS Code and Git for development and version control practices.
  • Provide hands-on experience with HTML, CSS, and GitHub for collaborative development.
  • Develop practical skills in creating structured, styled, and interactive web pages.


Syllabus

  1. Introduction to Web Technology

    This module covers the basics of web technology and full stack development, along with setting up VS Code, Git, and GitHub. It introduces HTML page structure, semantic elements, text formatting, links, lists, images, and multimedia, followed by practical implementation of all concepts.


  2. Web Designing

    This module covers creating and formatting HTML tables, including headers, captions, attributes, rowspan, colspan, nested tables, responsive design, and accessibility. It also introduces HTML forms with form elements, input types, labels, fieldsets, select menus, text areas, buttons, form attributes, and validation for better usability. Additionally, it explains the basics of Git, including its purpose, setup, repository creation, tracking changes, and using commands like init, add, commit, status, log, and .gitignore, along with practical exercises.


  3. Version Control with Git & GitHub and CSS Fundamentals

    This module covers creating a GitHub account and connecting local repositories for pushing, pulling, and cloning, along with branching, merging, handling conflicts, and collaborating using forks and pull requests. It also includes Git best practices and a full practical workflow. In CSS, it introduces selectors (element, class, ID), basic styling such as colors, fonts, alignment, and font sizes, and explains the box model with padding, margin, and borders, along with applying styles using inline, internal, and external CSS.


  4. CSS for Advanced Styling and Responsive Layout

    display: block, inline, inline-block, float and clear, Introduction to Flexbox: justify-content, align-items, flex- 10 direction, Practical in CSS, Units: px, %, em, rem, vh, vw, Media queries and breakpoints, Media queries and breakpoints, Mobile-first design, Responsive navigation concepts, Practicals, Project planning and design, Version control in large projects, Deploy using GitHub Pages, Final portfolio polishing, Project using HTML & CSS
